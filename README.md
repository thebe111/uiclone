UI Clone
=======

Discord UI Clone on VueJS

Getting Started
-------

* Install dependencies

```sh
$ npm install
```

* Compile and Hot-Reload for Development

```sh
$ npm run dev
```

* Type-Check, Compile and Minify for Production

```sh
$ npm run build
```

* Lint with [ESLint](https://eslint.org/)

```sh
$ npm run lint
```
